# Data Science Assignment: Trader Behavior & Sentiment Analysis

## Project Overview
This project explores how market sentiment (Fear vs. Greed) influences crypto trader behavior.  
Using:
1. `historical_data.csv` – detailed trade history from Hyperliquid.
2. `fear_greed_index.csv` – Bitcoin market sentiment classification.

We analyzed profitability, trade volume, and efficiency metrics, and identified patterns that can inform smarter trading strategies.

## How to Run the Notebook
1. Open `notebook_1.ipynb` in Google Colab.
2. Make sure both CSV files in `csv_files/` are available and loaded in the notebook.
3. Run all cells from top to bottom.
4. Visual outputs will be saved in the `outputs/` folder as `output1.png`, `output2.png`, etc.
